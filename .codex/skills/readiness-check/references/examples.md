# Examples

- **Proceed:** Recent load stable, sleep good, no soreness → `decision: proceed`, `modifications: []`.
- **Modify:** Slight soreness + high monotony → `decision: modify`, `modifications: ["swap to easy run", "cap at 30 minutes"]`.
- **Skip:** Pain or dizziness → `decision: skip`, `recommendation: "Stop and consult a qualified professional."`, `SafetyFlag { code: "injury_signal", severity: "high" }`.
