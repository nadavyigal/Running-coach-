• bg-gradient-to-br     → Bottom-right diagonal
• bg-gradient-to-r      → Right horizontal
• from-{color}-{shade}  → Gradient start
• via-{color}-{shade}   → Gradient middle
• to-{color}-{shade}    → Gradient end

Special Effects:
• backdrop-blur-sm      → Semi-transparent blur

• animate-pulse-glow    → Custom glow animation
• pointer-events-none   → SVG overlay passthrough
