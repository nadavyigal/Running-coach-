# Visual Clues Taxonomy for Geolocation

## Text-Based Indicators

### Language & Script
| Script | Primary Regions |
|--------|-----------------|
| Cyrillic | Russia, Ukraine, Bulgaria, Serbia, Mongolia |
| Arabic | Middle East, North Africa |
| Hebrew | Israel |
| Greek | Greece, Cyprus |
| Thai | Thailand |
| Japanese (mixed) | Japan |
| Korean (Hangul) | South Korea, North Korea |
| Simplified Chinese | China, Singapore |
| Traditional Chinese | Taiwan, Hong Kong |
| Devanagari | India, Nepal |

### Phone Number Formats
- +1 (XXX) XXX-XXXX → North America
- +44 XXXX XXXXXX → UK
- +49 XXX XXXXXXX → Germany
- +81 XX-XXXX-XXXX → Japan

### License Plates
- White/yellow front, white rear → UK, Ireland
- White only, EU blue strip → Continental Europe
- State/province specific → US, Canada, Australia
- Yellow background → Netherlands, Luxembourg

## Road Infrastructure

### Lane Markings
| Style | Regions |
|-------|---------|
| Yellow center, white edge | Americas, Taiwan |
| White throughout | Europe, UK, Australia |
| Yellow edge lines | Some Nordic countries |

### Traffic Light Styles
- Horizontal arrangement → US, Japan
- Vertical arrangement → Most of Europe
- Visor/hood style → Nordic countries, Canada
- LED matrix → Modern installations globally

### Road Signs
- Diamond warning signs → Americas
- Triangular warning signs → Europe, Asia
- Green highway signs → US
- Blue highway signs → UK, parts of Europe

## Architectural Indicators

### Building Materials
| Material | Common Regions |
|----------|----------------|
| Brick (red) | UK, Netherlands, Northern Europe |
| Stucco/plaster | Mediterranean, Latin America |
| Wood siding | Scandinavia, North America (suburban) |
| Concrete blocks | Southeast Asia, developing regions |

### Roof Styles
- Steep pitched, tile → Central Europe
- Flat roofs → Mediterranean, Middle East
- Metal roofing → Nordic, tropical regions
- Traditional tile (curved) → Spain, Portugal, Latin America

## Natural Environment

### Vegetation Zones
| Type | Indicators |
|------|------------|
| Tropical | Palm trees, broad leaves, dense canopy |
| Mediterranean | Olive trees, cypress, brown summers |
| Temperate | Deciduous mix, defined seasons |
| Boreal | Coniferous dominance, birch |
| Arid | Sparse, drought-adapted, sand visible |

### Soil & Terrain
- Red laterite → Tropical Africa, SE Asia, Brazil
- Black soil → Ukraine, US Great Plains
- Sandy → Coastal, desert regions
- Rocky/volcanic → Iceland, Hawaii, parts of Mediterranean

## Cultural & Commercial

### Global Brand Variants
- 7-Eleven (specific to region by store design)
- McDonald's (M logo universal, store design varies)
- Starbucks (presence limited in certain regions)

### Vehicle Indicators
- Right-hand drive → UK, Japan, Australia, India, SE Asia
- Left-hand drive → Americas, Continental Europe, China
- Domestic brands indicate region (Lada, Tata, Proton, etc.)

### Utility Infrastructure
| Feature | Region |
|---------|--------|
| Wooden power poles | Americas, parts of Asia |
| Concrete poles | Europe, developed Asia |
| Above-ground lines | Older areas, developing regions |
| Underground utilities | Modern European cities |
