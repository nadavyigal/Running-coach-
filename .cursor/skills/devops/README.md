# DevOps Skills

This directory is ready for DevOps and deployment-related skills.

## Recommended Skills to Install

### Sentry Skills
```bash
# Git workflow and code quality
git clone https://github.com/getsentry/sentry-skills.git
```

Skills include:
- `/commit` - Create commits with best practices
- `/create-pr` - Create pull requests
- `/code-review` - Perform code reviews
- `/find-bugs` - Identify code bugs

### Vercel Skills

# vercel-deploy-claimable - Deploy to Vercel
