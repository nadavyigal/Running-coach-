# Garmin Commercial Terms — RunSmart

Status: ANSWERED — email sent 2026-06-15, Garmin response received 2026-06-15.

## Questions sent

Q1. May RunSmart use Garmin-imported activity data inside a subscription product?
Q2. Are AI coaching insights derived from Garmin data permitted?
Q3. Do Health API metrics (sleep, HRV, stress, Body Battery, resting HR, Pulse Ox,
    respiration) carry license fees, revenue sharing, or usage caps in Production?
Q4. Is HISTORICAL_DATA_EXPORT (30-day maximum historical export) enabled by default
    in Production, or does it require a separate enablement request?
Q5. Is attribution required inside AI insights, run reports, App Store screenshots,
    or exported content? What is the approved attribution language?
Q6. May RunSmart use the phrase "Works with Garmin Connect" in App Store metadata?

## Garmin's answers

| Gate | Garmin answer | Status |
|---|---|---|
| Q1 — Subscription use | YES — RunSmart may use Garmin-imported activity data inside a subscription product. | Confirmed |
| Q2 — AI coaching from Garmin data | YES — AI coaching insights derived from Garmin data are permitted. | Confirmed |
| Q3 — License fees | NO — no license fees, revenue sharing, or Health API usage fees were stated. | Confirmed |
| Q4 — Historical data | 30 days max only. Historical export above 30 days is not available. | Confirmed limit |
| Q5 — Attribution requirements | Deferred to Garmin brand guidelines; compliance review required. | Pending brand review |
| Q6 — "Works with Garmin Connect" phrase | Deferred to Garmin brand guidelines; compliance review required. | Pending brand review |

## Decisions unblocked by each answer

| Answer | Unblocks |
|---|---|
| Q1 | Using Garmin in Pro subscription copy |
| Q2 | AI insights that use Garmin data |
| Q3 | Unit economics confirmation; paid positioning without Garmin API fees |
| Q4 | Marketing copy may mention recent history only up to 30 days |
| Q5 | Attribution wording in UI and screenshots remains blocked pending brand review |
| Q6 | "Works with Garmin Connect" in App Store remains blocked pending brand review |

## Safe defaults until brand review passes

| Claim | Safe default |
|---|---|
| "Works with Garmin Connect" in App Store | Do not use — use "Syncs with Garmin" or "Import from Garmin Connect" only if brand review allows |
| Garmin in Pro subscription copy | Allowed after Q1, but keep brand language conservative |
| AI insights citing Garmin by name | Allowed after Q2, but use "Based on your connected device data" until Q5 attribution language is approved |
| Historical data | Use "recent history" or "up to 30 days"; do not claim more than 30 days |
| Health API metrics as paid upsell | Commercially allowed after Q1-Q3, but brand/compliance review still applies |
